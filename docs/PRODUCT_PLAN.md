# JARVIS Life System

## Purpose
A gamified personal guidance system that helps me improve health, money, confidence, English, and technical skills.

## MVP Features
1. Daily missions
2. XP rewards
3. Levels
4. Streaks
5. Expense tracking
6. Weekly review
7. Simple guidance messages

## First Screen
Today’s Missions:
- Gym
- English Practice
- Track Expenses
- Learn .NET
- Eat Protein

## Rule
The app must always give one clear next action.
