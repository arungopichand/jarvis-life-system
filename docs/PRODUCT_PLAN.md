# Product Plan

## Product Thesis
JARVIS is a personal execution OS that turns intention into repeated, measurable action.

## Primary Outcomes
- Improve financial control and spending awareness
- Build consistent .NET/software engineering skill growth
- Strengthen English communication confidence
- Stabilize body systems (sleep, hydration, energy)
- Increase focus and reduce distraction-driven drift
- Build identity through repeated evidence, not motivation spikes

## Operating Modes
- Daily Mode: Today Command Center + execution + immediate feedback
- Review Mode: trend awareness + reflection
- Learning Mode: roadmap-driven guided skill growth
- Communication Mode: vocabulary + speaking confidence progression
- Writing Mode: diary reflection + blog draft output
- Setup Mode: configuration + template design

## Current Product Loops
1. Open Today Command Center for deterministic daily guidance
2. Execute mission + learning + communication reps
3. Log money systems (income-first, expenses secondary) and checklist evidence
4. Capture diary reflection and blog output
5. Receive context-aware assistant guidance based on daily guide + recent actions
6. Review weekly signals and continuity

## Planned Expansion
1. Advanced monthly review and trend diagnostics
2. AI-assisted coaching with deeper module memory and action sequencing
3. External API enrichment layer (optional) with resilient local fallback
4. Multi-device profile sync and optional auth
5. Long-horizon personal growth scoring and milestone system
