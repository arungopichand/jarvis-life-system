# MVP Scope

## In Scope
- Daily mission generation and completion
- Mission templates CRUD
- XP and completion progression signals
- Expense tracking with budget awareness
- Daily checklist persistence
- Streak and weekly stats calculations
- Mission history timeline
- Rule-based contextual assistant guidance
- Design-system governed UI with drift audit

## Out of Scope (Current)
- authentication and multi-user support
- cloud deployment pipelines
- AI inference backend for assistant responses
- advanced analytics and long-horizon forecasting
- formal notification/reminder system

## MVP Success Criteria
- User can run one daily execution loop end-to-end
- Data persists across sessions locally
- System always surfaces a practical next step
- UI remains calm, readable, and disciplined
